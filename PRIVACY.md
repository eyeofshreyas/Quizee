# Privacy Policy

## Overview

Quizee respects user privacy.

This project is developed for educational purposes and demonstrates secure authentication and user progress tracking.

---

## Information Collected

The application may store:

- Name
- Email address
- Encrypted password
- Quiz history
- Progress
- Scores
- XP
- Badges

Passwords are never stored in plain text.

---

## Data Storage

User information is stored in MongoDB.

Authentication uses JWT tokens.

---

## Data Usage

Collected data is used only for:

- Authentication
- Quiz functionality
- Progress tracking
- Leaderboards
- Badge generation

---

## Third-Party Services

This project does not intentionally share personal data with third-party services.

---

## User Responsibility

Users should:

- Protect their login credentials.
- Avoid sharing JWT tokens.
- Use strong passwords.

---

## Disclaimer

This project is intended for educational and portfolio purposes only.
