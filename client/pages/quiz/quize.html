15

Which AWS service is used to provide a managed DDoS protection service that safeguards applications running on AWS?

A
AWS Shield
B
AWS WAF
C
Amazon Inspector
D
AWS KMS
check_circle

Explanation

AWS Shield is a managed Distributed Denial of Service (DDoS) protection service that safeguards applications running on AWS. AWS Shield Standard provides protection against common infrastructure layer DDoS attacks for all AWS customers at no additional charge. For higher levels of protection, AWS Shield Advanced is available.
